下記のGitHubリポジトリから台湾華語の語彙リストをダウンロード。

https://github.com/ivankra/tocfl

Googleスプレッドシートにアップロードし、GASで毎日ランダムに単語とPinyinを取得、日本語訳も添えてDiscordに送信するようにした。

### コード

```js

const Words_num=7517;

function getData() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var num=Math.round(Math.random()*Words_num)+2;
  var word = sheet.getRange('B'+num).getValue();
  var pinyin = sheet.getRange('D'+num).getValue();
  
  discord_send(word+":"+pinyin+"\n \n"+(LanguageApp.translate(word, 'zh-TW', 'ja')));
}

```
