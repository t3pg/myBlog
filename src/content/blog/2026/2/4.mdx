# 挑戦MDX

ブログ記事にはMarkdown形式を使用しているが、MDXも使える。だから、記事ごとに自由な装飾や機能を実装することができる。
とりあえずボタンを追加してみた。  

<button>押せ!</button>

やり方は簡単で、以下のように書くだけだ。
```
<button>押せ!</button>
```

ここからさらに押されたときに何かが起こる、という機能を追加するにはもうひと手間必要なようだ。
